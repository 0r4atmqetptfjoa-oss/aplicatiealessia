# Profiles Module Assets

Acest director se ocupă de resursele pentru **Profiluri**. Poți include aici:

- Avatare prestabilite (imagini `.png`) pe care utilizatorii le pot selecta.
- Date de configurare sau preferințe (de exemplu, fișiere `.json` cu setările implicite pentru fiecare profil).
- O copie a pictogramei `ic_profiles.png` pentru personalizare.

Resursele pot fi accesate via `context.assets.open("modules/profiles/<fișier>")`.
