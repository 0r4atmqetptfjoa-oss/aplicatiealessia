# Upgrade Module Assets

Secțiunea **Upgrade** (Paywall) permite utilizatorului să deblocheze conținutul premium. Stochează aici:

- Elemente grafice folosite în ecranul de plată (de exemplu, imagini cu cadouri sau confetti).
- Texte localizate pentru prezentarea beneficiilor versiunii complete.
- Iconița de cadou (`ic_upgrade.png`) pentru editare.

Pentru acces, folosește `context.assets.open("modules/upgrade/<fișier>")`.
