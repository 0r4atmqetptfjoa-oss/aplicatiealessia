# Main Menu assets

Aici pui toate resursele specifice ecranului de **Main Menu**:
- `backgrounds/` – background static sau variantă fallback pentru cel actual (din `res/drawable`)  
- `ui/` – iconițe, logo, titlu static etc.

În plus, iconițele mari pentru module (Jocuri, Sunete, Stories etc.)
sunt în `assets/modules/<nume>/ic_*.png`.
