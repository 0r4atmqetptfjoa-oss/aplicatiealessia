# Asset Guidelines

This document outlines the structure and best practices for creating and managing assets in this project.

- **Sprites**: Animated objects and UI elements.
- **Icons**: UI icons and launcher icons.
- **Backgrounds**: Static backgrounds for menus and game levels.
- **Audio**: Music and sound effects.
- **Models**: 3D models.
- **Fonts**: Custom fonts.
- **Blender**: Source files and scripts for 3D modeling and rendering.
