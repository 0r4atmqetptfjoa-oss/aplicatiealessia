# Parental Gate Module Assets

Acest folder conține resursele pentru **Poarta Parinte** (Parental Gate). Poți stoca aici:

- Imagini pentru ecranele de verificare (de exemplu, ilustrații care să atragă atenția adultului).
- Baze de date cu întrebări sau exerciții pentru verificare (de ex. fișiere `.json` cu probleme aritmetice).
- O copie a pictogramei `ic_parental.png` pentru editare.

Pentru a accesa un fișier în cod, utilizează `context.assets.open("modules/parental_gate/<fișier>")`.
