# Settings Module Assets

Acest director conține resurse pentru secțiunea **Setări**. Aici poți plasa:

- Fișiere audio pentru sunete de butoane sau fundal.
- Configurații suplimentare în format `.json` pentru nivelul de dificultate, volum implicit etc.
- O copie a pictogramei `ic_settings.png` pentru personalizare.

Accesul la aceste fișiere se face prin `context.assets.open("modules/settings/<nume‑fișier>")`.
